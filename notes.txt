Directive
_________________

 ng-app, ng-model
 	-> directive qui injective directement des elements dans le html
 	-> ajouter des comportement sans ajouter de javascript

si directive qui commencent par "ng" -> native à angular
ng-app -> permet de dire que nous rentrons dans une application angular
ng-model -> on va dire que la value dans ce ng sera affichable dans une endroit du html
ng-init -> permet d'initier les valeurs à une div
ng-repeat -> correspond à un foreach classique php


Les filtres
_________________
  -> Permet de modifier des variables, des les filtrer et de les organiser
  -> s'applique de la facon suivante {{ variable | filtre }}
  -> possibilité d'ajout plusieurs filtre à la suite
  -> Pour l'order By, si "-" devant la variable, les données seront rangé dans l'ordre decroissant, et + pour ordre croissant
  -> pour preciser le type de flitre, on ecrit un objet ou l'on definit sur qu'elle element le filtre s'applique
      / filter : {content : query} = ici content correspondra a la valeur ng-model query


Controller et Scope
____________________

Scope -> permet de faire la liaison entre le controller et la view
Controller -> n'importe quel nom, par convention fini par controller, et en entré, on met $scope


Les Modules
_____________

  -> Les modules permettent d'organiser notre application via angular js
  -> Mettre les différentes librairies ou script dans un module que l'on peut decider de charger selon la page
  -> app =  angular.module('MonApp', []);
    / Permet de créer le module de l'application
  -> Quand le module est créer, il faut bien le definir dans le ng-app
    / ng-app="nomDuModule"
  -> Lorsqu'on crée un module, nous avons un []
    / Il permet de gérer les dépendances
    / Si on load une dépendance angular commme ça / <script src=".../angular-select.js"
    / Le resultat est le suivant angular.module('MonApp', [angular-select]);
