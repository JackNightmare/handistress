Tracer un itinéraire Optimisé

Handistress vous permet de tracer des itinéraires prenant en compte votre handicape.

Pour cela, vous devez renseigner les accès auxquels vous pouvez accèder (escaliers, ascensseurs, escalators ...). L'application calculera alors le meilleur chemin pour vous.

Vous pouvez à tout moment changer les informations vous concernant, ce qui impactera les futurs itinéraires

Ajouter des lieux et des accès

Avec Handistress, vous pouvez contribuer aux informations présentes sur la carte. Nous utilisons votre position géographiques afin d'enregister les markers sur la carte.

Vous pouvez ajouter un marker de type "Lieu" (restaurant, ecoles, boutiques ... ). Ces markers demandent de nombreuses informations afin de connaitre aux mieux l'accessibilité du lieu en question. Nous demandons donc de renseigner des informations générales comme le type de lieu, le nom et une description, mais en plus de cela, les accès s'y trouvant (escaliers, ascenseurs ...) et des informations complémentaires tel que des descriptions des entrées et sorties, si il y a des toilettes adaptés etc...

Il est aussi possible de créer de marker de type "Accès" (escaliers, escalators, rampes ...). Ces derniers permettent de référencer les accès se situant dans les rues. Ces derniers ne demandent très peu d'information, juste un nom et une description en plus du type d'accès.
